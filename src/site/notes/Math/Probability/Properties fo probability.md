#probability 
1. $P(A^{c)} = 1 - P(A)$, where $A^c$ is A complement
2. If $A \subseteq B$, then $P(A) \le P(B)$
3. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
	1. $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) + P(A \cap B \cap C)$
