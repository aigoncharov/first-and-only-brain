#probability 

## Expected value (mean)
### Discrete
$$E(X) = \sum\limits_{j = 1}^{\infty} x_{j} P(X = x_{j})$$
### Linearity of expectation
$$E(X + Y) = E(X) + E(Y)$$
$$E(cX) = cE(x)$$
**Holds for dependent RVs!**
